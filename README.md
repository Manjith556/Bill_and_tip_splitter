# Bill & Tip Splitter

## Overview
Bill & Tip Splitter is a responsive web application that helps users quickly calculate tips and split the total bill among multiple people. The application provides instant calculations with real-time updates, making it simple and convenient to manage shared expenses.

## Features
- Calculate tip amount instantly
- Split the total bill among multiple people
- Real-time calculation updates
- Responsive and mobile-friendly design
- Simple and intuitive user interface
- Input validation for accurate calculations

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## How to Use
1. Enter the total bill amount.
2. Select or enter the desired tip percentage.
3. Enter the number of people sharing the bill.
4. View the tip amount, total bill, and amount per person instantly.
